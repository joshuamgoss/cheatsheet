Starting a new project

From the command line:
  1. rails new <project name>
  2. cd <project name>
  3. git innit
  4. git commit -m "initial commit"

Scaffolding CRUD for a new model

rails generate scaffold <model name, capitalized and singular> <column name, singular, lower case>:<data type> <column name, singular, lower case>:<data type> example: rails g scaffold User name:string email:string administrator:boolean hireddate:datetime

Data types

  • binary
  • boolean
  • date
  • datetime
  • decimal
  • float
  • integer
  • bigint
  • primary_key
  • references
  • string
  • text
  • time
  • timestamp
Full documentation available at the official rails website.
